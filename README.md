# OpenMaya Stub Generator

Generate python stub files for Autodesk OpenMaya modules.

> ℹ️ As of right now this generator only generates stubs for the OpenMaya2 _(`api`)_ modules.

## What are stub files?
Stub files _(`.pyi`)_ provide type information for Python modules, enabling better IDE support with auto-completion, type checking, and docstrings.
